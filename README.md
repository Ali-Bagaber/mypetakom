# MyPetakom

MyPetakom is a web-based management system developed using Laravel, PHP, and MySQL.

## Technologies
- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User management
- Dashboard modules
- Data handling and system workflows
- Web-based interface for managing system information

## Purpose
The project was developed as a web-based system project to manage and organize data through multiple modules and dashboards.
